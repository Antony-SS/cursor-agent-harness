# Scratch

Informal working area for **session context**, not the product contract.

**Agent policy:** Nothing here should be written unless **you explicitly ask** the AI to persist chat or discussion context in this folder—otherwise it stays empty. Keeps notes from getting long uninvited.

Use it for anything that should not live in `/docs` yet (or ever):

- Notes from back-and-forth with the user  
- Tentative decisions and options under discussion  
- Feature ideas / backlog snippets  
- Copy-paste context, links, or reminders for the next session  

**`/docs` stays canonical** for what the system *is* and *should do*. When something in scratch is agreed, promote it into the right `/docs` file (often at wrap-up—see `.cursor/rules/02-definition-of-done.mdc`).

Add any filenames you like under this folder (e.g. `notes.md`, `backlog.md`). Nothing here is required by the harness.
